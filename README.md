# Snake Game em Rust

Projeto desenvolvido para aprendizado da linguagem Rust.

## Sobre

Este repositório contém uma implementação do clássico jogo Snake utilizando Rust e a biblioteca Piston Window. O objetivo principal é explorar os conceitos fundamentais da linguagem, incluindo:

- Sistema de ownership e borrowing
- Pattern matching
- Traits e generics
- Structs e enums
- Gerenciamento de memória

## Como executar

```bash
cd snake
cargo run
```

## Controles

- **Setas direcionais**: Movimentar a cobra
- **ESC**: Sair do jogo
